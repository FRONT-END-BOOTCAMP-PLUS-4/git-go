# 🚀 GITGO

### **회고를 일상처럼, 기록을 습관처럼**

![image](https://github.com/user-attachments/assets/17b8c618-a133-40dc-8390-b84101024a71)

**깃고(Gitgo)는**  
개발자가 자신의 커밋과 PR을 기반으로 자연스럽게 회고를 작성하고  
기록할 수 있도록 돕는 **개인 회고 기록 서비스**입니다.
